In this section our goal is to answer: "Can we stimate the price of a car based on its characteristics?", we are going to go through this question using differentes tools along the journey.

The problem

We need to understand what features or characteristics make a car valuable or what determine its price, we can also compare data which other selling cars or car´s that were sold in the past. We have a lot of information to start dig around.

The first thing we need to do is Understand our Data, what we got? which format it is?, what kind of information each row give us, trough this example we are going to work with a csv file, which is comma separated value, that means each row of the data is actually one record to be aware of. each columns give us some kind of information and we can relate with documentation a priori.

We also notice column 26 is "Price" that is our actual target value, or label, this means, price is the value that we want to predict from the dataset, and the predictors should be all the others variables listed.

Importing and Exporting Data

Importing is the process of loading data from different sources using Python.
When we load new data into our enviroment we need to keep in mind two factors:

  • Format: Some common format are csv,json, xlslsx, hdf
  • Path File: Tell us where data is stored, it could be locally or on the internet
In Python the way to import these data is through Pandas using read_csv 

import pandas as pd

                  url = "https://archive.ics.uci.edu/ml/machine-learningdatabases/autos/imports-85.data"
                  
                  df = pd.read_csv(url)

Our Data has no columns header, so we need to specify to our program that. 

import pandas as pd

                  url = "https://archive.ics.uci.edu/ml/machine-learningdatabases/autos/imports-85.data"
                  
                  df = pd.read_csv(url, header=None)

It is recommend to check out the data to get familiar with, using df is not recommended because it use a lot of computing resources, instead we can use df.head(n) where n are n first rows of our dataset. Similarly, df.tail(n) retrieves us the n last rows of the dataset.

It is too difficult to work without headers in our dataset, we can assign it then, we create an array within all the headers associated with the data and then we assign it using df.columns = headers

After working with your dataset, maybe you want to save it locally in same format as you load it, you can do this using the method df.to_csv(path)

Getting Started Analyzing Data

Before any analysis you need to understand your data, what datatypes are in the data, how the data is distributed along the dataset, taking into account these things you maybe point out potential issues, which may need to be resolved later on.
Data has variety of types, the would object, strings, int, float, etc.

In pandas, we use df.types to check datatypes of all columns, we need to make sure all columns datatypes make sense, and later on try to solve it, because some methods depends on the datatype they rely.

The first Insight of our dataset maybe is df.describe which retrieve a statistical summary of each colum, telling us information like, mean, std, min_vale, max_value, etc.

This df.describe() only works with numeric data, columns or features that are non-numeric will be skipped, but this is not a issue at all, if we want all the information including non-numeric data, we need to add a parameter inside our method, so if we type df.describe(include="all") we retrieve all the information both numeric and non-numeric.

Another useful method is df.info() which retrieves concise summary of our dataset with the first and last 30 rows of data.
